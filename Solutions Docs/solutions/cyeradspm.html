The Cyera DSPM data connector allows you to connect to your Cyera's DSPM instance and ingesting Classifications, Assets, Issues, and Identity Definitions into Microsoft Sentinel. The data connector is built on Microsoft Sentinel's Codeless Connector Platform and uses the Cyera's API to fetch Cyera's DSPM Telemetry once recieced can be correlated with security events creating custom columns so that queries don't need to parse it again, thus resulting in better performance.
