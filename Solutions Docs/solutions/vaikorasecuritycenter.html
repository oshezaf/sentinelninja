The Vaikora Microsoft Defender for Cloud solution integrates Vaikora AI-driven security signal detection with Microsoft Sentinel and Microsoft Defender for Cloud. A Logic App playbook polls the Vaikora API every 6 hours, filters high-severity actions and behavioral anomalies, and writes them to a custom Log Analytics table (Vaikora_AgentSignals_CL). Analytic rules then surface these signals in Sentinel for investigation.
