The Airlock Digital Solution for Microsoft Sentinel enables you to ingest Airlock Digital application control and execution logs into Microsoft Sentinel using the Codeless Connector Platform (CCP). This solution provides visibility into file executions, server activities, and security event summaries from your Airlock Digital server.

Underlying Microsoft Technologies used:

This solution takes a dependency on the following technologies, and some of these dependencies either may be in Preview state or might result in additional ingestion or operational costs:

a. Codeless Connector Platform (CCP)
