The Amazon Web Services (AWS) Identity and Access Management (IAM) Solution for Microsoft Sentinel allows you to manage resources in AWS via playbooks thats use the AWS IAM API. The Playbboks included in the solution allow Enriching Incident with user information add tag to a user in AWS and delete access keys for users.
