CyberArk Enterprise Password Vault generates an xml Syslog message for every action taken against the Vault. The EPV will send the xml messages through the Microsoft Sentinel.xsl translator to be converted into CEF standard format and sent to a syslog staging server of your choice (syslog-ng, rsyslog). The Log Analytics agent installed on your syslog staging server will import the messages into Microsoft Log Analytics. Refer to the CyberArk documentation for more guidance on SIEM integrations.

Additional Information

🛠️ Device Configuration: On the EPV, configure dbparm.ini to send syslog messages in CEF format to the proxy machine on port 514 TCP.
