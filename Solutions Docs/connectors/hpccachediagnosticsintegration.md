[Azure HPC Cache](https://learn.microsoft.com/en-us/azure/hpc-cache/hpc-cache-overview) is a file caching service for high-performance computing workloads.

To collect Azure HPC Cache logs into a Log Analytics workspace, configure [Azure Diagnostic Settings](https://learn.microsoft.com/en-us/azure/hpc-cache/hpc-cache-monitor) for the resource, selecting the log categories to send to the workspace.
