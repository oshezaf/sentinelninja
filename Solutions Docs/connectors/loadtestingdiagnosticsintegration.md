[Azure Load Testing](https://learn.microsoft.com/en-us/azure/load-testing/overview-what-is-azure-load-testing) is a fully managed load-testing service.

To collect Azure Load Testing logs into a Log Analytics workspace, configure [Azure Diagnostic Settings](https://learn.microsoft.com/en-us/azure/load-testing/how-to-monitor-load-testing) for the resource, selecting the log categories to send to the workspace.
