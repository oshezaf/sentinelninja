[CyberArk Privileged Access Manager (PAM)](https://www.cyberark.com/products/privileged-access-manager/) is a privileged access management solution that secures, controls, and monitors privileged accounts, credentials, and sessions across the enterprise.

The connector uses the Azure Monitor Agent (AMA) to collect CEF logs, routing them via a Data Collection Rule (DCR) to the CommonSecurityLog table ([learn more about CEF via AMA](https://learn.microsoft.com/azure/sentinel/cef-syslog-ama-overview)).

To collect CyberArk PAM logs into Microsoft Sentinel:
- On the EPV, configure `dbparm.ini` to send syslog messages in CEF format to the AMA collector on port 514 TCP.
- The PAM will use the `Microsoft Sentinel.xsl` translator to convert XML messages to CEF format. Refer to the [CyberArk SIEM integration documentation](https://docs.cyberark.com/privilege-cloud-standard/Latest/en/Content/Privilege%20Cloud/privCloud-connect-siem.htm) for detailed guidance.
