[Votiro](https://www.votiro.com/) is a content disarm and reconstruction (CDR) platform that sanitizes files and documents by neutralizing malware and zero-day threats while preserving full file functionality.

The connector uses the Azure Monitor Agent (AMA) to collect CEF logs, routing them via a Data Collection Rule (DCR) to the CommonSecurityLog table ([learn more about CEF via AMA](https://learn.microsoft.com/azure/sentinel/cef-syslog-ama-overview)).

To collect Votiro logs into Microsoft Sentinel, configure Votiro Endpoints to send syslog messages in CEF format to the AMA collector on port 514 TCP. See [Configure Votiro](https://learn.microsoft.com/azure/sentinel/unified-connector-cef-device#votiro).

## Tables Ingested

This connector ingests data into the following tables:

| Table | Selection Criteria | Transformations | Ingestion API | Lake-Only |
|:------|:-------------|:---------------:|:-------------:|:---------:|
| [`CommonSecurityLog`](../tables/commonsecuritylog.md) | `DeviceProduct == "Votiro cloud"`<br>`DeviceVendor == "Votiro"` | ✓ | ✓ | ✓ |
