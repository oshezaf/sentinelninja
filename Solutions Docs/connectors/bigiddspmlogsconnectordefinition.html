The BigID DSPM data connector provides the capability to ingest BigID DSPM cases with affected objects and datasource information into Microsoft Sentinel.

Permissions

Resource Provider Permissions: - Workspace (Workspace): Read and Write permissions are required.

Custom Permissions: - BigID DSPM API access: Access to the BigID DSPM API through a BigID Token is required.
