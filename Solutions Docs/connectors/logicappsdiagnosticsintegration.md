[Azure Logic Apps](https://learn.microsoft.com/en-us/azure/logic-apps/logic-apps-overview) is a cloud platform for automating workflows with little to no code.

To collect Azure Logic Apps logs into a Log Analytics workspace, configure [Azure Diagnostic Settings](https://learn.microsoft.com/en-us/azure/logic-apps/monitor-logic-apps-log-analytics) for the resource, selecting the log categories to send to the workspace.
