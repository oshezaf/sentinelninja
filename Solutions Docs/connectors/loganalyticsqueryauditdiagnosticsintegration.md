[Log Analytics query audit](https://learn.microsoft.com/en-us/azure/azure-monitor/logs/query-audit) records details of every query executed against a Log Analytics workspace, including the user, query text, execution time, and resource consumption.

To collect query audit logs into a Log Analytics workspace, configure [Azure Diagnostic Settings](https://learn.microsoft.com/azure/azure-monitor/logs/query-audit#configure-query-auditing) on the Log Analytics workspace resource, enabling the Audit log category.
