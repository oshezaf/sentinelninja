[WatchGuard Firebox](https://www.watchguard.com/wgrd-products/firewall-appliances) is a network security appliance that provides firewall, VPN, intrusion prevention, application control, and threat detection for enterprise networks.

The connector uses the Azure Monitor Agent (AMA) to collect Syslog messages, routing them via a Data Collection Rule (DCR) to the Syslog table ([learn more about Syslog via AMA](https://learn.microsoft.com/azure/sentinel/cef-syslog-ama-overview)). For more information, see https://www.watchguard.com/wgrd-products/cloud-and-virtual-firewalls.

To collect WatchGuard Firebox logs into Microsoft Sentinel, configure the Firebox to send syslog messages to the AMA collector. For more information, see the [WatchGuard cloud and virtual firewalls documentation](https://www.watchguard.com/wgrd-products/cloud-and-virtual-firewalls).
