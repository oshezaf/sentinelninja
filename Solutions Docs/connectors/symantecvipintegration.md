[Symantec VIP (Validation and ID Protection)](https://vip.symantec.com/) is a cloud-based multi-factor authentication service that adds an extra layer of security by requiring users to verify their identity using one-time passwords or push notifications.

The connector uses the Azure Monitor Agent (AMA) to collect Syslog messages, routing them via a Data Collection Rule (DCR) to the Syslog table ([learn more about Syslog via AMA](https://learn.microsoft.com/azure/sentinel/cef-syslog-ama-overview)).

To collect Symantec VIP logs into Microsoft Sentinel, [follow these instructions](https://aka.ms/sentinel-symantecvip-configurationsteps) to configure Symantec VIP Enterprise Gateway to forward syslog messages to the AMA collector.
