[Azure Chaos Studio](https://learn.microsoft.com/en-us/azure/chaos-studio/chaos-studio-overview) is a managed service for improving resilience through chaos engineering experiments.

To collect Azure Chaos Studio logs into a Log Analytics workspace, configure [Azure Diagnostic Settings](https://learn.microsoft.com/en-us/azure/chaos-studio/chaos-studio-run-experiment) for the resource, selecting the log categories to send to the workspace.
