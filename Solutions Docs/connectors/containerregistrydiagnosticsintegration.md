[Azure Container Registry](https://learn.microsoft.com/en-us/azure/container-registry/container-registry-intro) is a managed container registry for storing and managing container images.

To collect Azure Container Registry logs into a Log Analytics workspace, configure [Azure Diagnostic Settings](https://learn.microsoft.com/en-us/azure/container-registry/monitor-service) for the resource, selecting the log categories to send to the workspace.
