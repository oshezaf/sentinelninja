[Performance counters](https://learn.microsoft.com/en-us/azure/azure-monitor/agents/data-collection-performance) provide real-time measurements of hardware and software component utilization on Windows and Linux machines, including CPU, memory, disk, and network metrics.

To collect performance counters, create a [data collection rule](https://learn.microsoft.com/azure/azure-monitor/agents/data-collection-performance) with the Azure Monitor Agent, selecting the performance counter categories and sampling intervals to monitor.
