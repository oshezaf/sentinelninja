[Azure Network Watcher](https://learn.microsoft.com/en-us/azure/network-watcher/connection-monitor-overview) is a network monitoring and diagnostic service.

To collect connection monitoring data, configure [Connection Monitor](https://learn.microsoft.com/azure/network-watcher/connection-monitor-overview) in Azure Network Watcher.
