[Event Tracing for Windows (ETW)](https://learn.microsoft.com/en-us/azure/azure-monitor/agents/data-collection-etw) is a Windows kernel-level tracing facility that captures detailed diagnostic events from applications and system components.

To collect ETW events, create a [data collection rule](https://learn.microsoft.com/azure/azure-monitor/agents/data-collection-etw) with the Azure Monitor Agent, specifying the ETW providers and event levels to capture.
