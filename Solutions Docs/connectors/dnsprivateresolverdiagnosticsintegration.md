[Azure DNS Private Resolver](https://learn.microsoft.com/en-us/azure/dns/dns-private-resolver-overview) is a service for resolving DNS queries from and to on-premises environments.

To collect Azure DNS Private Resolver logs into a Log Analytics workspace, configure [Azure Diagnostic Settings](https://learn.microsoft.com/azure/azure-monitor/essentials/create-diagnostic-settings) for the resource, selecting the log categories to send to the workspace.
