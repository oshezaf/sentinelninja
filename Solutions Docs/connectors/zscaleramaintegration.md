[Zscaler Internet Access (ZIA)](https://www.zscaler.com/products/zscaler-internet-access) is a cloud-native secure web gateway that provides inline inspection of all internet traffic for threat protection, data loss prevention, and access control.

The connector uses the Azure Monitor Agent (AMA) to collect CEF logs, routing them via a Data Collection Rule (DCR) to the CommonSecurityLog table ([learn more about CEF via AMA](https://learn.microsoft.com/azure/sentinel/cef-syslog-ama-overview)).

To collect Zscaler ZIA logs into Microsoft Sentinel, configure Zscaler to send syslog messages in CEF format to the AMA collector on port 514 TCP. See the [Zscaler Microsoft Sentinel integration guide](https://aka.ms/ZscalerCEFInstructions).

## Tables Ingested

This connector ingests data into the following tables:
