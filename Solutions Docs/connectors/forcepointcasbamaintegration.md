[Forcepoint CASB](https://www.forcepoint.com/product/casb-cloud-access-security-broker) is a cloud access security broker that provides visibility and control over cloud application usage, enabling organizations to monitor user activities, enforce data security policies, and detect threats across SaaS environments.

The connector uses the Azure Monitor Agent (AMA) to collect CEF logs, routing them via a Data Collection Rule (DCR) to the CommonSecurityLog table ([learn more about CEF via AMA](https://learn.microsoft.com/azure/sentinel/cef-syslog-ama-overview)).

For additional vendor documentation, see the [Forcepoint CASB Integration Guide](https://frcpnt.com/casb-sentinel).

## Tables Ingested

This connector ingests data into the following tables:

| Table | Selection Criteria | Transformations | Ingestion API | Lake-Only |
|:------|:-------------|:---------------:|:-------------:|:---------:|
| [`CommonSecurityLog`](../tables/commonsecuritylog.md) | `DeviceVendor == "Forcepoint CASB"` | ✓ | ✓ | ✓ |
