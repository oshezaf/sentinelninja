[Sysmon for Linux](https://github.com/Sysinternals/SysmonForLinux) is a system monitoring tool that provides detailed information about process creations, network connections, file changes, and other system events on Linux hosts. This solution depends on ASIM to work as expected ([deploy ASIM](https://aka.ms/DeployASIM)).

The connector uses the Azure Monitor Agent (AMA) to collect Syslog messages, routing them via a Data Collection Rule (DCR) to the Syslog table ([learn more about Syslog via AMA](https://learn.microsoft.com/azure/sentinel/cef-syslog-ama-overview)).

## Tables Ingested

This connector ingests data into the following tables:

| Table | Selection Criteria | Transformations | Ingestion API | Lake-Only |
|:------|:-------------|:---------------:|:-------------:|:---------:|
| [`Syslog`](../tables/syslog.md) | `ProcessName == "sysmon"` | ✓ | ✓ | ✓ |
