# RFI-search-workforce-user

This playbook searches the Recorded Future Identity Intelligence Module for compromised workforce users. This playbook depends on: - RFI-add-EntraID-security-group-user - RFI-confirm-EntraID-risky-user - RFI-lookup-and-save-user Those playbooks need to be installed **manually** before installing current playbook.
