[Azure Storage Mover](https://learn.microsoft.com/en-us/azure/storage-mover/overview) is a managed migration service for moving files and folders to Azure Storage.

To collect Azure Storage Mover logs into a Log Analytics workspace, configure [Azure Diagnostic Settings](https://learn.microsoft.com/en-us/azure/storage-mover/log-monitoring) for the resource, selecting the log categories to send to the workspace.
