[GitLab](https://about.gitlab.com/solutions/devops-platform/) is a comprehensive DevOps platform that provides source code management, CI/CD pipelines, security scanning, and project management in a single application (GitLab Enterprise Edition - Standalone).

The connector uses the Azure Monitor Agent (AMA) to collect Syslog messages, routing them via a Data Collection Rule (DCR) to the Syslog table ([learn more about Syslog via AMA](https://learn.microsoft.com/azure/sentinel/cef-syslog-ama-overview)).
