[Arista Awake Security](https://www.arista.com/en/solutions/network-detection-and-response) is a network detection and response (NDR) platform that uses AI to detect and respond to threats across managed and unmanaged devices on the network.

The connector uses the Azure Monitor Agent (AMA) to collect CEF logs, routing them via a Data Collection Rule (DCR) to the CommonSecurityLog table ([learn more about CEF via AMA](https://learn.microsoft.com/azure/sentinel/cef-syslog-ama-overview)).

To collect Arista Awake Security logs into Microsoft Sentinel:
- In the Awake Security Platform, navigate to Detection Management Skills > Add New Skill.
- Set the Expression to forward CEF events to the AMA collector on port 514. See [Configure Arista Awake Security](https://learn.microsoft.com/azure/sentinel/unified-connector-cef-device#aristaawakesecurity) for detailed steps.

## Tables Ingested
