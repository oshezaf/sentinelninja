[Cisco Secure Email Gateway (SEG)](https://www.cisco.com/c/en/us/products/security/email-security/index.html) is an email security solution that protects against phishing, malware, spam, and data loss in inbound and outbound email communications.

The connector uses the Azure Monitor Agent (AMA) to collect CEF logs, routing them via a Data Collection Rule (DCR) to the CommonSecurityLog table ([learn more about CEF via AMA](https://learn.microsoft.com/azure/sentinel/cef-syslog-ama-overview)).

To collect Cisco SEG logs into Microsoft Sentinel:
- Configure a [Log Subscription](https://www.cisco.com/c/en/us/td/docs/security/esa/esa14-0/user_guide/b_ESA_Admin_Guide_14-0/b_ESA_Admin_Guide_12_1_chapter_0100111.html#con_1134718) on the Cisco SEG appliance.
- Select Consolidated Event Logs in the Log Type field.
- Forward the logs in CEF format to the AMA collector on port 514.
