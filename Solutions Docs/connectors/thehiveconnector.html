The TheHive data connector provides the capability to ingest TheHive security incident response platform data into Microsoft Sentinel through the REST API. Refer to API documentation for more information. The connector provides the ability to get cases, tasks, and alerts from TheHive and visualize them in Azure Sentinel.

Permissions

Resource Provider Permissions: - Workspace (Workspace): Read and Write permissions are required.

Custom Permissions: - TheHive API access: TheHive API Version 4 and above access is required for the TheHive API.
