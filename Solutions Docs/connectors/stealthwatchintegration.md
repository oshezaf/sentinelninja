[Cisco Secure Cloud Analytics](https://www.cisco.com/c/en/us/products/security/stealthwatch/index.html) (formerly Stealthwatch) is a network detection and response (NDR) solution that uses behavioral analytics and machine learning to detect threats across on-premises, cloud, and hybrid environments.

The connector uses the Azure Monitor Agent (AMA) to collect Syslog messages, routing them via a Data Collection Rule (DCR) to the Syslog table ([learn more about Syslog via AMA](https://learn.microsoft.com/azure/sentinel/cef-syslog-ama-overview)).
