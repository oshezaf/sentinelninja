[Juniper SRX](https://www.juniper.net/us/en/products-services/security/srx-series/) is a security gateway and firewall platform that provides firewall, intrusion prevention, VPN, and unified threat management for enterprise and service provider networks.

The connector uses the Azure Monitor Agent (AMA) to collect Syslog messages, routing them via a Data Collection Rule (DCR) to the Syslog table ([learn more about Syslog via AMA](https://learn.microsoft.com/azure/sentinel/cef-syslog-ama-overview)).

To collect Juniper SRX logs into Microsoft Sentinel, configure Juniper SRX to forward syslog messages to the AMA collector: [Traffic Logs](https://kb.juniper.net/InfoCenter/index?page=content&id=KB16509&actp=METADATA) and [System Logs](https://kb.juniper.net/InfoCenter/index?page=content&id=kb16502).
