[Trend Micro Deep Security](https://www.trendmicro.com/en_us/business/products/hybrid-cloud/deep-security.html) is a server security platform that provides anti-malware, intrusion prevention, firewall, integrity monitoring, and log inspection for physical, virtual, and cloud server workloads.

The connector uses the Azure Monitor Agent (AMA) to collect CEF logs, routing them via a Data Collection Rule (DCR) to the CommonSecurityLog table ([learn more about CEF via AMA](https://learn.microsoft.com/azure/sentinel/cef-syslog-ama-overview)).

To collect Trend Micro Deep Security logs into Microsoft Sentinel:
- Configure Deep Security to forward events to the AMA collector using CEF format. See [this knowledge article](https://aka.ms/Sentinel-trendmicro-kblink) and [configuration instructions](https://aka.ms/Sentinel-trendMicro-connectorInstructions).
- Save the [TrendMicroDeepSecurity](https://aka.ms/TrendMicroDeepSecurityFunction) KQL function.
