[PostgreSQL](https://www.postgresql.org/) is an open-source object-relational database system known for its reliability, feature robustness, extensibility, and standards compliance.

To collect PostgreSQL logs into Microsoft Sentinel:
- Configure PostgreSQL with `log_destination='stderr'` and `logging_collector=on`. See [PostgreSQL logging docs](https://www.postgresql.org/docs/current/runtime-config-logging.html).
- Configure a Data Collection Rule for the `PostgreSQL_CL` table. Log locations — Windows: `C:\*.log`, Linux: `/var/log/*.log`. See [Custom Logs via AMA configuration](https://learn.microsoft.com/azure/sentinel/unified-connector-custom-device#postgresql-events).
