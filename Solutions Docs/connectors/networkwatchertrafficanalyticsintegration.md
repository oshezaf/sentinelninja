[Azure Network Watcher](https://learn.microsoft.com/en-us/azure/network-watcher/traffic-analytics) is a network monitoring and diagnostic service.

To collect traffic data, enable [Traffic Analytics](https://learn.microsoft.com/azure/network-watcher/traffic-analytics) on your Network Security Groups (NSGs).
