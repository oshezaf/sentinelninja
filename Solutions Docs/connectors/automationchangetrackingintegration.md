[Azure Automation](https://learn.microsoft.com/en-us/azure/automation/change-tracking/overview) is a cloud-based service for process automation, configuration management, and update management.

To collect change tracking data, enable [Change Tracking and Inventory](https://learn.microsoft.com/azure/automation/change-tracking/overview) for the machine using the Azure Monitor Agent.
