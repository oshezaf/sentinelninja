[Azure Container Instances](https://learn.microsoft.com/en-us/azure/container-instances/container-instances-overview) is a service for running containers without managing servers.

To collect Azure Container Instances logs into a Log Analytics workspace, configure [Azure Diagnostic Settings](https://learn.microsoft.com/en-us/azure/container-instances/container-instances-log-analytics) for the resource, selecting the log categories to send to the workspace.
