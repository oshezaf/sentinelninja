[OSSEC](https://www.ossec.net/) is an open-source host-based intrusion detection system (HIDS) that performs log analysis, integrity checking, rootkit detection, time-based alerting, and active response.

The connector uses the Azure Monitor Agent (AMA) to collect CEF logs, routing them via a Data Collection Rule (DCR) to the CommonSecurityLog table ([learn more about CEF via AMA](https://learn.microsoft.com/azure/sentinel/cef-syslog-ama-overview)).

To collect OSSEC logs into Microsoft Sentinel, [configure syslog output](https://www.ossec.net/docs/docs/manual/output/syslog-output.html) on the OSSEC server to forward events in CEF format to the AMA collector on port 514.

## Tables Ingested

This connector ingests data into the following tables:
