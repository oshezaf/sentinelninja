Data Collection Rules - Security Event logs

Enable data collection rule for Security Logs Security Events logs are collected only from Windows agents. 1. Add chosen DCs on Resources tab. 2. Select Security log level

Common level is the minimum required. Please select 'Common' or 'All Security Events' on DCR definition. - Create data collection rule
