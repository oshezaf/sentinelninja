[VMware ESXi](https://www.vmware.com/products/esxi-and-esx.html) is a bare-metal hypervisor that provides a virtualization platform for running multiple virtual machines on a single physical server.

The connector uses the Azure Monitor Agent (AMA) to collect Syslog messages, routing them via a Data Collection Rule (DCR) to the Syslog table ([learn more about Syslog via AMA](https://learn.microsoft.com/azure/sentinel/cef-syslog-ama-overview)).

To collect VMware ESXi logs into Microsoft Sentinel, configure ESXi syslog forwarding to the AMA collector. See [VMware ESXi 3.5 and 4.x](https://kb.vmware.com/s/article/1016621) or [VMware ESXi 5.0+](https://docs.vmware.com/en/VMware-vSphere/5.5/com.vmware.vsphere.monitoring.doc/GUID-9F67DB52-F469-451F-B6C8-DAE8D95976E7.html).
