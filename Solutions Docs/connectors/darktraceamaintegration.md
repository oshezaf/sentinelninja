[Darktrace](https://darktrace.com/) is an AI-powered cyber defense platform that uses self-learning models to detect, investigate, and respond to cyber threats in real time across cloud, email, network, and endpoint environments.

The connector uses the Azure Monitor Agent (AMA) to collect CEF logs, routing them via a Data Collection Rule (DCR) to the CommonSecurityLog table ([learn more about CEF via AMA](https://learn.microsoft.com/azure/sentinel/cef-syslog-ama-overview)).

To collect Darktrace logs into Microsoft Sentinel:
- In the Darktrace Threat Visualizer, navigate to System Config > Modules > Microsoft Sentinel.
- Configure the server address (AMA collector IP) and port 514.
- Enable Send Alerts and save.

## Tables Ingested

This connector ingests data into the following tables:

| Table | Selection Criteria | Transformations | Ingestion API | Lake-Only |
|:------|:-------------|:---------------:|:-------------:|:---------:|
| [`CommonSecurityLog`](../tables/commonsecuritylog.md) | `DeviceVendor == "Darktrace"` | ✓ | ✓ | ✓ |
