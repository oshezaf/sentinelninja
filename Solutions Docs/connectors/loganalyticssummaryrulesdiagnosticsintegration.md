[Log Analytics summary rules](https://learn.microsoft.com/en-us/azure/azure-monitor/logs/summary-rules) aggregate data from a Log Analytics workspace into a summary table on a scheduled basis, reducing data volume while preserving key insights for long-term analysis.

To collect summary rule execution logs into a Log Analytics workspace, configure [Azure Diagnostic Settings](https://learn.microsoft.com/azure/azure-monitor/essentials/create-diagnostic-settings) on the Log Analytics workspace resource, enabling the summary logs category.
