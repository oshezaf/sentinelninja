[ISC BIND](https://www.isc.org/bind/) is the most widely used Domain Name System (DNS) software on the internet, providing authoritative DNS server and recursive resolver services.

The connector uses the Azure Monitor Agent (AMA) to collect Syslog messages, routing them via a Data Collection Rule (DCR) to the Syslog table ([learn more about Syslog via AMA](https://learn.microsoft.com/azure/sentinel/cef-syslog-ama-overview)).

For additional vendor documentation, see [ISC BIND DNS Logs](https://kb.isc.org/docs/aa-01526).
