Connects to the UniFi Site Manager API and ingests four data types: Sites, Devices, Hosts, and ISP Metrics.

Prerequisites:

  1. Sign in to UniFi Site Manager

  2. Navigate to API in the left navigation

  3. Click Create API Key

  4. Copy and securely store the generated key (shown only once)
