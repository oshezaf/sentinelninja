[Cisco Unified Computing System (UCS)](https://www.cisco.com/c/en/us/products/servers-unified-computing/index.html) is a data center platform that integrates compute, networking, storage, and virtualization resources into a cohesive system.

The connector uses the Azure Monitor Agent (AMA) to collect Syslog messages, routing them via a Data Collection Rule (DCR) to the Syslog table ([learn more about Syslog via AMA](https://learn.microsoft.com/azure/sentinel/cef-syslog-ama-overview)).

To collect Cisco UCS logs into Microsoft Sentinel, [configure Cisco UCS to forward syslog](https://www.cisco.com/c/en/us/support/docs/servers-unified-computing/ucs-manager/110265-setup-syslog-for-ucs.html#configsremotesyslog) using the IP address or hostname for the Linux device with the AMA agent installed.
