[Pulse Connect Secure](https://www.pulsesecure.net/products/pulse-connect-secure/) (now Ivanti Connect Secure) is an SSL VPN solution that provides secure remote access to enterprise applications and resources for mobile and remote users.

The connector uses the Azure Monitor Agent (AMA) to collect Syslog messages, routing them via a Data Collection Rule (DCR) to the Syslog table ([learn more about Syslog via AMA](https://learn.microsoft.com/azure/sentinel/cef-syslog-ama-overview)). For setup guidance, see [Follow the instructions](https://help.ivanti.com/ps/help/en_US/PPS/9.1R13/ag/configuring_an_external_syslog_server.htm).

To collect Pulse Connect Secure logs into Microsoft Sentinel, [follow the instructions](https://help.ivanti.com/ps/help/en_US/PPS/9.1R13/ag/configuring_an_external_syslog_server.htm) to configure an external syslog server to forward logs to the AMA collector.
