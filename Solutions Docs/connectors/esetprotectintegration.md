[ESET PROTECT](https://www.eset.com/int/business/endpoint-security/) is a centralized security management platform that provides endpoint protection, including antivirus, firewall, and EDR capabilities across Windows, macOS, Linux, and mobile devices.

The connector uses the Azure Monitor Agent (AMA) to collect Syslog messages, routing them via a Data Collection Rule (DCR) to the Syslog table ([learn more about Syslog via AMA](https://learn.microsoft.com/azure/sentinel/cef-syslog-ama-overview)).

To collect ESET PROTECT logs into Microsoft Sentinel:
- Configure [syslog server settings](https://help.eset.com/protect_admin/latest/en-US/admin_server_settings_syslog.html) in ESET PROTECT.
- [Configure export to syslog](https://help.eset.com/protect_admin/latest/en-US/admin_server_settings_export_to_syslog.html) to forward events to the AMA collector. See also [security considerations](https://learn.microsoft.com/en-us/azure/sentinel/connect-log-forwarder?tabs=rsyslog#security-considerations) for the log forwarder.
