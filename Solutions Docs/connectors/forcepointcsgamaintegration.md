[Forcepoint Cloud Security Gateway (CSG)](https://www.forcepoint.com/product/cloud-security-gateway) is a converged cloud security service that provides visibility, control, and threat protection for users and data across web and email channels.

The connector uses the Azure Monitor Agent (AMA) to collect CEF logs, routing them via a Data Collection Rule (DCR) to the CommonSecurityLog table ([learn more about CEF via AMA](https://learn.microsoft.com/azure/sentinel/cef-syslog-ama-overview)). For more information, see https://www.forcepoint.com/product/cloud-security-gateway.

For additional vendor documentation, see the [Forcepoint CSG Integration Guide](https://frcpnt.com/csg-sentinel).

## Tables Ingested

This connector ingests data into the following tables:

| Table | Selection Criteria | Transformations | Ingestion API | Lake-Only |
|:------|:-------------|:---------------:|:-------------:|:---------:|
| [`CommonSecurityLog`](../tables/commonsecuritylog.md) | `DeviceProduct in "Email,Web"`<br>`DeviceVendor == "Forcepoint CSG"` | ✓ | ✓ | ✓ |
