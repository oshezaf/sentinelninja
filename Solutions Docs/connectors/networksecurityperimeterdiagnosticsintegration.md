[Azure Network Security Perimeter](https://learn.microsoft.com/en-us/azure/private-link/network-security-perimeter-concepts) is a network security boundary for Azure PaaS resources.

To collect Azure Network Security Perimeter logs into a Log Analytics workspace, configure [Azure Diagnostic Settings](https://learn.microsoft.com/en-us/azure/private-link/network-security-perimeter-diagnostic-logs) for the resource, selecting the log categories to send to the workspace.
