[Infoblox Network Identity Operating System (NIOS)](https://www.infoblox.com/glossary/network-identity-operating-system-nios/) is a DDI (DNS, DHCP, and IPAM) platform that provides core network services, network automation, and visibility into network activity.

The connector uses the Azure Monitor Agent (AMA) to collect Syslog messages, routing them via a Data Collection Rule (DCR) to the Syslog table ([learn more about Syslog via AMA](https://learn.microsoft.com/azure/sentinel/cef-syslog-ama-overview)).

To collect Infoblox NIOS logs into Microsoft Sentinel, [configure syslog](https://www.infoblox.com/wp-content/uploads/infoblox-deployment-guide-slog-and-snmp-configuration-for-nios.pdf) on the Infoblox NIOS appliance to forward logs to the AMA collector.
