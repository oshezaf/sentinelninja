1. Option 1: Logic app

  1. Create a new logic app Use When a HTTP request is recived as the Trigger and save it. It will now have generated a URL that can be used in the ProActive webconsole configuration. Add an action: Select the Azure Log Analytics Data Collector and choose Send Data Enter Connection Name, Workspace ID and Workspace Key, you will find the information needed in your Log Analytics workspace under Settings-->Agents-->Log Analytics agent instructions. In JSON Request body add @triggerBody(). in Custom Log Name add Netclean_Incidents.
