[Squid Proxy](http://www.squid-cache.org/) is an open-source caching and forwarding HTTP web proxy that supports HTTP, HTTPS, and FTP, widely used for web traffic filtering, access control, and accelerating network performance.

To collect Squid Proxy logs into Microsoft Sentinel, configure a Data Collection Rule for the `SquidProxy_CL` table. Log locations — Windows: `C:\Squid\var\log\squid\*.log`, Linux: `/var/log/squid/*.log`. See [Custom Logs via AMA configuration](https://learn.microsoft.com/azure/sentinel/unified-connector-custom-device#squidproxy).
