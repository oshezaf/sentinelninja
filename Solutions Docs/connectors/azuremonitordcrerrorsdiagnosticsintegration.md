[Data collection rules (DCR)](https://learn.microsoft.com/en-us/azure/azure-monitor/essentials/data-collection-rule-overview) define how data is collected and transformed by the Azure Monitor pipeline, including specifications for data sources, transformations, and destinations.

To collect DCR error logs into a Log Analytics workspace, configure [Azure Diagnostic Settings](https://learn.microsoft.com/azure/azure-monitor/essentials/create-diagnostic-settings) on the data collection rule resource, enabling the error log category.
