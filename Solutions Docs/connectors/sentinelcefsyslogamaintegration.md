[Microsoft Sentinel](https://learn.microsoft.com/en-us/azure/sentinel/connect-cef-syslog-ama) is a cloud-native SIEM and SOAR solution.

To collect these logs, configure a data collection rule using the Azure Monitor Agent (AMA). See [Microsoft Sentinel documentation](https://learn.microsoft.com/en-us/azure/sentinel/connect-cef-syslog-ama) for setup instructions.
