[Azure Relay](https://learn.microsoft.com/en-us/azure/azure-relay/relay-what-is-it) is a service for establishing secure connections between on-premises and cloud applications.

To collect Azure Relay logs into a Log Analytics workspace, configure [Azure Diagnostic Settings](https://learn.microsoft.com/en-us/azure/azure-relay/diagnostic-logs) for the resource, selecting the log categories to send to the workspace.
