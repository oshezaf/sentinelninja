[Azure Arc-enabled Kubernetes](https://learn.microsoft.com/en-us/azure/azure-arc/kubernetes/overview) is a service that extends Azure management to Kubernetes clusters running anywhere.

To collect Azure Arc-enabled Kubernetes logs into a Log Analytics workspace, configure [Azure Diagnostic Settings](https://learn.microsoft.com/en-us/azure/azure-arc/kubernetes/monitor-gitops-flux-2) for the resource, selecting the log categories to send to the workspace.
