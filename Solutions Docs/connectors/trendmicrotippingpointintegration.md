[Trend Micro TippingPoint](https://www.trendmicro.com/en_us/business/products/network/intrusion-prevention/tipping-point-threat-protection-system.html) is a network-based intrusion prevention system (IPS) that provides real-time threat protection by inspecting network traffic and blocking attacks inline.

The connector uses the Azure Monitor Agent (AMA) to collect CEF logs, routing them via a Data Collection Rule (DCR) to the CommonSecurityLog table ([learn more about CEF via AMA](https://learn.microsoft.com/azure/sentinel/cef-syslog-ama-overview)).

To collect Trend Micro TippingPoint logs into Microsoft Sentinel, configure TippingPoint SMS to send syslog messages in ArcSight CEF Format v4.2 to the AMA collector on port 514 TCP.
