[Palo Alto Networks Cortex Data Lake (CDL)](https://www.paloaltonetworks.com/cortex/cortex-data-lake) is a cloud-based logging and analytics service that collects, transforms, and integrates network security data from Palo Alto Networks firewalls and Prisma Access.

The connector uses the Azure Monitor Agent (AMA) to collect CEF logs, routing them via a Data Collection Rule (DCR) to the CommonSecurityLog table ([learn more about CEF via AMA](https://learn.microsoft.com/azure/sentinel/cef-syslog-ama-overview)).

To collect Palo Alto CDL logs into Microsoft Sentinel, [configure log forwarding](https://docs.paloaltonetworks.com/cortex/cortex-data-lake/cortex-data-lake-getting-started/get-started-with-log-forwarding-app/forward-logs-from-logging-service-to-syslog-server.html) from Cortex Data Lake to the AMA collector in CEF format.

For additional vendor documentation, see [PAN-OS Syslog Field Descriptions](https://docs.paloaltonetworks.com/pan-os/11-1/pan-os-admin/monitoring/use-syslog-for-monitoring/syslog-field-descriptions) and [Palo Alto CEF Configuration Guides](https://docs.paloaltonetworks.com/resources/cef).

## Tables Ingested

This connector ingests data into the following tables:

| Table | Selection Criteria | Transformations | Ingestion API | Lake-Only |
|:------|:-------------|:---------------:|:-------------:|:---------:|
| [`CommonSecurityLog`](../tables/commonsecuritylog.md) | `DeviceProduct == "LF"`<br>`DeviceVendor == "Palo Alto Networks"` | ✓ | ✓ | ✓ |
