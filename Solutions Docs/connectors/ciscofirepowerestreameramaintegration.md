[Cisco Firepower](https://www.cisco.com/c/en/us/products/security/firepower-ngfw/index.html) is a next-generation firewall (NGFW) and intrusion prevention system (IPS) that provides advanced threat detection, prevention, and network security intelligence. The eStreamer client-server API enables real-time event streaming in CEF format.

The connector uses the Azure Monitor Agent (AMA) to collect CEF logs, routing them via a Data Collection Rule (DCR) to the CommonSecurityLog table ([learn more about CEF via AMA](https://learn.microsoft.com/azure/sentinel/cef-syslog-ama-overview)).

To collect Cisco Firepower logs into Microsoft Sentinel:
- Install and configure the [Firepower eNcore eStreamer client](https://www.cisco.com/c/en/us/td/docs/security/firepower/670/api/eStreamer_enCore/eStreamereNcoreSentinelOperationsGuide_409.html).
- Configure the client to forward events in CEF format to the AMA collector on port 514.

For additional vendor documentation, see [Cisco Firepower Threat Defense Syslog Messages](https://www.cisco.com/c/en/us/td/docs/security/firepower/Syslogs/b_fptd_syslog_guide.html) and the [Microsoft Sentinel Cisco firewalls connector guide](https://learn.microsoft.com/azure/sentinel/cisco-ftd-firewall).
