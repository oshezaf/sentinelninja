## Schema (16 columns)

**Source:** [Connector definition](https://github.com/Azure/Azure-Sentinel/blob/master/Solutions/AbnormalSecurity%5CData%20Connectors%5CAbnormalSecurity_CCF/AbnormalSecurity_table_Remediation.json)

| Column Name | Type | Description |
|:------------|:-----|:------------|
| abx_body | string | Remediation action event payload |
| abx_body_abx_body_action_s | string | Remediation action taken (e.g. MESSAGE_MOVED_TO_TRASH) |
| abx_body_abx_body_initiated_by_s | string | Who initiated the remediation (e.g. auto, user) |
| abx_body_abx_body_recipient_s | string | Email address of the affected recipient |
| abx_body_abx_body_remediation_id_s | string | Unique remediation action identifier |
| abx_body_abx_body_status_s | string | Status of the remediation action |
| abx_body_abx_body_threat_id_s | string | Associated threat identifier |
| abx_body_abx_metadata_event_type_s | string | From abx_body.abx_metadata |
| abx_body_abx_metadata_timestamp_s | string | From abx_body.abx_metadata |
| abx_body_abx_metadata_trace_id_g | string | From abx_body.abx_metadata |
| abx_metadata | string | Full event metadata object containing event_type, timestamp, and trace_id |
| abx_metadata_event_type_s | string | Event type identifier from top-level abx_metadata |
| abx_metadata_timestamp_s | string | Event timestamp from top-level abx_metadata |
| abx_metadata_trace_id_g | string | Trace ID (GUID) from top-level abx_metadata |
| Time | string | Event timestamp in RFC3339 format |
