Elastic Load Balancer distributes incoming traffic across multiple instances in multiple availability Zones. This increases the fault tolerance of your applications.  Unwanted changes to Elastic Load Balancer specific security groups could open your environment to attack and  hence needs monitoring.  More information: https://aws.amazon.com/elasticloadbalancing/.
