HYASInsight-IP-C2-Attribution

author: Paul van Gool, HYAS Infosec

This playbook uses the HYAS Insight connector to automatically enrich incidents generated by Sentinel with C2-Attribution information. You need a valid subscription in order to use the connector and playbook. Learn more about the integration via the connector documentation or visit HYAS Insight to request a trial key.
